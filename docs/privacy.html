SmartBookmark Pro는 확장 프로그램의 기능 제공을 위해 아래 정보를 수집합니다. 모든 정보는 사용자의 기기 내부에만 저장됩니다.

데이터 유형 수집 내용 저장 위치

방문 기록 방문 URL, 방문 횟수, 최근 방문 시간 Chrome Storage (로컬)
유튜브 시청 기록 동영상 제목, URL, 채널명, 시청 시간, 카테고리 IndexedDB (로컬)
탭 세션 저장된 탭 그룹(URL, 제목) Chrome Storage (로컬)

2 정보 저장 방식

🗄️ Chrome Storage (chrome.storage.local)

설정, 북마크, 방문기록 캐시 등을 저장합니다. Chrome 브라우저 내부에만 유지되며 동기화(sync)는 사용하지 않습니다.

💾 IndexedDB

대용량 유튜브 시청기록을 저장합니다. 브라우저 샌드박스 내에 유지되며 외부 접근이 불가합니다.

🔐 암호화 저장

보안 잠금 비밀번호와 AI API 키는 PBKDF2(반복 60만 회) + AES-GCM 암호화를 적용하여 저장합니다.

3 권한 사용 목적

SmartBookmark Pro가 요청하는 Chrome 권한과 그 사용 이유입니다.

권한 사용 목적
bookmarks 북마크 읽기, 생성, 편집, 삭제 및 폴더 관리
history 방문 기록 조회 및 자동 북마크 추천 분석
storage 사용자 설정 및 데이터 로컬 저장
tabs 탭 세션 저장/복원, 현재 탭 정보 읽기
activeTab 활성 탭의 URL 및 제목 읽기 (북마크 추가 시)
scripting 페이지 하이라이트 기능 실행 (사용자 요청 시만)
sidePanel 브라우저 사이드패널 대시보드 표시
alarms 정기적 데이터 정리 및 추천 갱신
contextMenus 우클릭 메뉴를 통한 빠른 북마크 추가

4 외부 서비스 연동 (AI 기능)

⚠️
AI 기능은 기본 비활성화 상태입니다. 사용자가 직접 API 키를 입력하고 기능을 활성화해야만 작동합니다.

AI 기능 사용 시, 입력하신 API 키를 통해 아래 외부 서비스에 요청이 전송될 수 있습니다.

  • OpenAI API (api.openai.com) — GPT 모델 사용 시
  • Anthropic API (api.anthropic.com) — Claude 모델 사용 시
  • Google Gemini API (generativelanguage.googleapis.com) — Gemini 모델 사용 시
  • Ollama (localhost) — 로컬 모델 사용 시 (외부 전송 없음)
  • 사용자 지정 엔드포인트 — 사용자가 직접 설정한 서버

전송되는 데이터는 AI 분류 또는 요약을 요청하는 북마크/페이지 정보에 한하며, 각 서비스의 개인정보 처리방침이 적용됩니다. SmartBookmark Pro는 해당 데이터를 수집하거나 저장하지 않습니다.

5 광고 처리

SmartBookmark Pro는 무료 사용자에게 서비스 유지를 위한 네이티브 광고를 표시할 수 있습니다.

  • 광고는 사용자 개인정보를 기반으로 타겟팅하지 않습니다
  • 광고 식별자(IDFA, GAID 등)를 수집하지 않습니다
  • 광고 클릭 여부 데이터를 외부로 전송하지 않습니다
  • Pro 플랜 사용자는 광고가 표시되지 않습니다

6 데이터 보안

🔒 로컬 전용 저장

모든 데이터는 사용자 기기의 Chrome 브라우저 내부에만 저장됩니다. 인터넷 연결 없이도 모든 핵심 기능이 동작합니다.

🔑 비밀번호/패턴 보호

보안 잠금 기능 설정 시 비밀번호는 PBKDF2(600,000회 반복) + AES-GCM으로 암호화되어 저장됩니다. 원문 복구가 불가능합니다.

🌐 네트워크 격리

확장 프로그램은 SmartBookmark 개발자 서버와 어떠한 통신도 하지 않습니다. AI 기능은 사용자가 직접 설정한 API 서비스에만 연결됩니다.

7 데이터 삭제

아래 방법으로 저장된 데이터를 삭제할 수 있습니다.

  • 개별 데이터 삭제: 확장 프로그램 설정 → 데이터 관리 탭에서 항목별 삭제 가능
  • 전체 초기화: 설정 → 데이터 관리 → "모든 데이터 초기화" 버튼 클릭
  • 확장 프로그램 제거 시: Chrome에서 확장 프로그램 삭제 시 모든 저장 데이터가 자동으로 완전 삭제됩니다

8 아동 개인정보 보호

SmartBookmark Pro는 만 14세 미만 아동을 대상으로 하지 않으며, 고의적으로 아동의 개인정보를 수집하지 않습니다.

9 방침 변경 안내

본 개인정보 처리방침이 변경될 경우, 이 페이지의 "최종 업데이트" 날짜를 갱신하고 확장 프로그램 업데이트 노트를 통해 안내합니다. 중요한 변경 사항은 확장 프로그램 내 알림으로도 안내됩니다.
